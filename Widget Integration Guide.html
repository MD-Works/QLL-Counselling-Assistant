Warm & Nurturing

Calm & Professional

Security note
About the Groq API key
Placing the API key in a data-* attribute means it is visible in the page source. For a low-traffic practice like QLL this is acceptable — the Groq free tier has generous rate limits that make abuse unlikely.

When you are ready to secure the key properly, the solution is a Cloudflare Worker proxy: the widget calls the Worker, the Worker holds the key server-side and calls Groq, and the key is never exposed in the browser. This is Phase 2 of the project roadmap.
